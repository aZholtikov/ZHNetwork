# Сеть на основе ESP-NOW для ESP8266
Очень маленькая и простая библиотека для построения сети на основе ESP-NOW для устройств ESP8266.

## Функции:

1. Максимальный размер передаваемых данных - 200 байт. В настоящее время только незашифрованные сообщения.
2. Все узлы не видны сетевому сканеру.
3. Не требуется предварительное соединение для передачи данных.
4. Широковещательная или целевая передача данных.

## Примечание:

API см. в файле ZHNetwork.h

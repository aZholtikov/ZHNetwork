# Mesh сеть на основе ESP-NOW для ESP8266/ESP8285
Библиотека для построения Mesh сети на основе ESP-NOW для устройств ESP8266/ESP8285.

## Функции:

1. Максимальный размер передаваемых данных - 200 байт. В настоящее время только незашифрованные сообщения.
2. Все узлы не видны сетевому сканеру.
3. Не требуется предварительное соединение для передачи данных.
4. Широковещательная или целевая передача данных.
5. Отсутствуют какие-либо периодические/синхронизационные сообщения в сети. Все устройства в "тихом режиме" и не "фонят" в эфир.
6. Каждый узел имеет свою независимую таблицу маршрутизации, обновляемую только по мере возникновения необходимости.
7. Каждый узел получит сообщение, если он "видит" хотя бы одно устройство в сети.
8. Количество устройств в сети и площадь использования не ограничены (гипотетически). Протестировано на 30 устройствах в пределах одного дома/участка.

## Тестирование:

1. Запрограммируйте 2 приемника и 1 передатчик (в коде передатчика укажите MAC 1-го приемника).
2. Подключите 1-й приемник к компьютеру. Включите монитор серийного порта. Включите передатчик. Приемник начнет принимать данные.
3. Перемещайте передатчик как можно дальше от приемника, до тех пор пока приемник не сможет принимать данные (заэкранируйте при необходимости). З.Ы. При тестировании я помещал приемник и передатчик в алюминиевые банки на расстоянии примерно 15 метров. Этого хватало для нарушения передачи.
4. Включите 2-й приемник и поместите его между 1-ым приемником и передатчиком (желательно по середине). 1-й приемник возобновит прием данных (с ретрансляцией через 2-й приемник). З.Ы. Вместо 2-го приемника можно использовать передатчик - роли не играет.
5. Вуаля. ;-)
